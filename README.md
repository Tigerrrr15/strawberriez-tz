Описание проекта 
Название проекта: StrawberrieZ
Назначение: Веб-приложение для просмотра и фильтрации товаров интернет-магазина, а также управления ассортиментом со стороны администратора.
Целевая аудитория:
•	Покупатели — ищут товары, используют фильтры, добавляют в корзину.
•	Администраторы / контент-менеджеры — наполняют каталог, управляют категориями и остатками.
Ключевая функциональность:
•	Просмотр каталога товаров в виде сетки карточек.
•	Фильтрация по цене, категории, бренду, размеру, сезону.
•	Поиск по названию с подсказками.
•	Сортировка по цене, новизне, популярности.
•	Детальная страница товара с галереей, выбором размера, описанием.
•	Корзина с возможностью изменения количества и применения промокода.
•	Административная панель для управления товарами и категориями.

Макеты интерфейса
Главная страница / Каталог
 
Описание:
Слева блок фильтров (цена, категории, размер). Справа карточки товаров в сетке 3-4 колонки. Вверху строка поиска, иконка корзины с количеством товаров и иконка с профилем пользователя.
Карточка товара 
Описание экрана:
Страница карточки товара открывается при клике на товар из каталога. Верхняя часть содержит «хлебные крошки» - навигационную цепочку, по которой можно вернуться в категорию или на главную. Левая половина экрана отведена под галерею изображений: крупное главное фото и миниатюры для переключения. 
Правая половина содержит всю информацию о товаре:
•	Название и артикул.
•	Рейтинг и количество отзывов (кликабельно и ведёт к отзывам).
•	Текущая цена, старая цена (если есть скидка) и размер экономии.
•	Статус наличия с точным количеством на складе.
•	Блок выбора размера - размеры отображаются в виде кнопок. Доступные размеры активны, недоступные - заблокированы и визуально отличаются.
•	Счётчик количества с кнопками «+» и «-», минимальное значение 1, максимальное — остаток на складе.
•	Две кнопки действий: основная зелёная «Добавить в корзину» и дополнительная «Купить в один клик» (сразу переходит к оформлению).
Ниже расположены вкладки с подробной информацией: описание, технические характеристики, размерная сетка, отзывы, условия доставки. Это позволяет не перегружать основной блок, но даёт доступ ко всем данным на одной странице.
Админ-панель (список товаров)
Описание экрана:
Админ-панель предназначена для сотрудников магазина, которые управляют ассортиментом. Интерфейс выполнен в строгом, функциональном стиле, ориентирован на скорость работы.
Верхняя панель:
•	Крупная кнопка «+ Добавить товар» - открывает форму создания нового товара.
•	Кнопки быстрого перехода в разделы управления категориями и брендами.
Блок фильтрации и поиска:
•	Позволяет быстро находить товары по различным критериям:
•	Выбор категории из выпадающего списка.
•	Фильтр по наличию (все товары, только в наличии, только отсутствующие).
•	Поиск по названию или артикулу (частичное совпадение).
•	Кнопка «Сбросить» очищает все фильтры.
Таблица товаров (колонки):
•	ID (внутренний идентификатор).
•	Фото (миниатюра изображения).
•	Название (кликабельно, ведёт на страницу редактирования).
•	Категория (отображает принадлежность).
•	Цена (с возможностью быстрого редактирования (двойной клик)).
•	Остаток (текущее количество на складе).
•	Статус «Активен» (товар виден на сайте) / «Неактивен» (скрыт).
•	Действия иконки: ✎ (редактировать), 🗑 (удалить), ⚡ (быстрое изменение цены/остатка).
Пагинация: При большом количестве товаров таблица разбивается на страницы. Можно выбрать количество записей на странице (25, 50, 100).
Блок статистики: В нижней части отображается сводная информация по каталогу: общее количество товаров, наличие, отсутствие, товары с низким остатком. Это помогает администратору быстро оценить состояние каталога и принять меры (например, заказать поставку).
Описание API
Формат данных: JSON
Аутентификация: JWT (для админ-эндпоинтов)
GET /products
Получение списка товаров с фильтрацией, сортировкой и пагинацией.
Пример:
Json

{
  "data": [
    {
      "id": 101,
      "name": "Клубника квадратная игрушка",
      "price": 1990,
      "old_price": 2990,
      "image": " https://img.freepik.com/premium-photo/isolated-strawberry-single-strawberry-fruit-isolated-white-background-with-clipping-path-image_120872-32587.jpg ",
      "in_stock": true,
      "rating": 4.7
    }
  ],
  "meta": {
    "total": 245,
    "page": 1,
    "limit": 24,
    "pages": 11
  }
}

GET /categories
Получение дерева категорий.
Пример:
json
[
  {
    "id": 1,
    "name": "Детские игрушки",
    "slug": "toys",
    "children": [
      {
        "id": 2,
        "name": " Клубника квадратная игрушка ",
        "slug": "strawberry",
        "children": []
      }
    ]
  }
]

Эндпоинты корзины:
•	GET /cart - Получение содержимого корзины.
•	POST /cart/items - Добавление товара в корзину.

Тело запроса:

json
{
  "product_id": 101,
  "quantity": 1,
  "size": "30"
}

•	PUT /cart/items/{id} - Изменение количества товара.
•	DELETE /cart/items/{id} - Удаление товара из корзины.
